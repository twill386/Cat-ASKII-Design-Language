
==========================
CADL Traits
==========================

This file provides the traits a cat can be assigned as well
as the options for each trait. This information will be used 
for the interpreter as well as the ASCII rendering file.

-------------------------------------
 TRAIT: ears
-------------------------------------
Values:
    - pointy
    - droopy
    - round
    - long
    - short 

-------------------------------------
 TRAIT: mouth
-------------------------------------
Values: 
    - smile
    - frown
    - kiss
    - scowl
    - open
    - smirk

-------------------------------------
 TRAIT: body
-------------------------------------
Values:
    - smooth
    - fluffy
    - normal 
    - chubby

-------------------------------------
 TRAIT: tail
-------------------------------------
Values:
    - fluffy
    - straight
    - curled 

-------------------------------------
 TRAIT: mood
-------------------------------------
Values:
    - happy 
    - sleepy
    - excited
    - loving
    - curious
    - angry  
    - sad 
Note:
    Mood affects the eyes of the cat.
    If only a mood is selected and no
    other traits a preset built cat
    will be used.     

-------------------------------------
 TRAIT: whiskers
-------------------------------------
Values:
    - long
    - short
    - curled
